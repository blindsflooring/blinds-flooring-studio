🔒 Role Permissions GuideOwner eyes only
OwnerFull control — you
  • See ALL jobs across ALL branches
  • Create, edit and DELETE any job
  • Change payments and amounts
  • Access Admin Panel and all reports
  • Add/manage users and branches
ManagerBranch manager / senior staff
  • See ALL jobs across ALL branches
  • Create and edit any job
  • Cannot DELETE jobs
  • Can capture and update payments
  • Access reports and conversions
  • Cannot access Admin Panel
RepSales rep / estimator
  • See only their assigned branch jobs
  • Create new quotes and jobs
  • Edit only their own jobs
  • Cannot DELETE jobs
  • Cannot change payment amounts
  • Can see their own conversion stats
AdminOffice admin / bookkeeper
  • See jobs across all branches (read)
  • Capture and update payments
  • Update job status
  • Cannot create new quotes
  • Cannot DELETE jobs
  • Access audit log
⚠ Tip: Use Manager for Ryno and Madri if you want them to see and work on jobs from both branches when you are not around.
